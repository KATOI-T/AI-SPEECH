# 機能一覧

各機能の概要です。

| ID | 機能 | 概要 |
|----|------|------|
| F-001 | 3Dモデル表示 | VRM / GLB の3DモデルをThree.jsで表示する |
| F-002 | リップシンク | Viseme（口形素）を VRM の BlendShape にマッピングし、発話に合わせて口を動かす |
| F-003 | アニメーション機能 | 感情状態に応じたアニメーションクリップをバックエンドAPIで管理・提供する |
| F-004 | 音声入力（STT） | マイク音声を Azure Speech 等で文字化する（プロバイダー切替可） |
| F-005 | 音声出力（TTS） | テキストを音声合成し、リップシンク用の Viseme も併せて生成する |
| F-006 | AI会話生成 | OpenAI + LangChain で文脈を踏まえた応答と感情を生成する |
| F-007 | シナリオ管理 | ロールプレイのシナリオ（状況・目標・評価基準）を CRUD する |
| F-008 | キャラクター管理 | キャラクターの人格・3Dモデル・音声設定を CRUD し、モデルをアップロードする |
| F-009 | S3ストレージ移管 | ユーザーがアップロードした3Dモデルを S3 に保管できるようにする（任意） |
| F-009 | セッション管理 | Redis で会話状態・履歴・ターン数・有効期限を管理する |
| F-010 | 会話アニメーション | LLM が返す感情を3レイヤーで合成し、自然なキャラクターの動きを実現する |
| F-011 | アニメーション制御拡張 | 挨拶動作の改善、表情連動、会話中の細粒度アニメーション制御 |
| F-012 | 背景画像設定 | 会話画面の背景画像を設定できるようにする |
| F-013 | アニメーション反映の信頼性改善 | 会話アニメーションが確実に反映されるよう描画処理を改善する |
| F-014 | シナリオ・キャラクターAI自動生成 | AI を用いてシナリオやキャラクター設定を自動生成する |
